# ChatGPT Quick Opener (Chrome Extension)

Macのショートカットキー（`Command` + `Option` + `G`）で、瞬時にChatGPT (https://chatgpt.com/) を新しいタブで開くChrome拡張機能です。

## 特徴

- **Manifest V3 対応**: 最新のChrome拡張機能仕様に準拠しています。
- **軽量**: バックグラウンドで待機し、ショートカット入力時のみ動作します。

## インストール方法

この拡張機能はストア公開されていないため、以下の手順でChromeに読み込ませる必要があります。

1. このリポジトリをZIPでダウンロードして解凍するか、`git clone` します。
2. Chromeブラウザを開き、アドレスバーに `chrome://extensions/` と入力して移動します。
3. 画面右上の **「デベロッパーモード」** をONにします。
4. 左上の **「パッケージ化されていない拡張機能を読み込む」** をクリックします。
5. 手順1で用意したフォルダを選択します。

## ⚠️ 重要：ショートカットキーの設定（必ず行ってください）

現時点では、拡張機能のインストールのみではショートカットはうまく作動しません。以下の手順に従い設定を完了させてください。

1. Chromeのアドレスバーに以下を入力して開きます。
`chrome://extensions/shortcuts`
2. リストの中から **「ChatGPT QA」** を探します。
3. 鉛筆アイコン（または入力欄）をクリックし、キーボードで実際に **`Command` + `Shift` + `G`** を押して登録します。
4. とはいえ、ここで自由に設定できます。あくまで上記の構成は一例なのでお好みのショートカットを作ってください。

これで設定は完了です。

## 使い方

- 任意のページを見ている状態で **`Command` + `Shift` + `G`** を押すと、ChatGPTが新しいタブで開きます。

## ファイル構成

- `manifest.json`: 設定ファイル
- `background.js`: タブを開く処理を行うスクリプト
